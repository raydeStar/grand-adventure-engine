Operator Notes

Smoke flow Create or seed a player, run look, stats, inventory, help, then confirm the room and story state update together.
Exploration flow Move north to force room generation, then inspect the new room and verify the world map expands.
Repeatability Use explicit player ids in the portal workshop so automation and manual admin checks hit the same actors every time.
Protected API Use the same admin login in the browser, Playwright, or PowerShell to run mutations and direct command probes through the secured endpoints.

Quick Workflow

Send Message / Announcement

Send a message directly to a player's Discord thread, or broadcast to everyone.

Warp Player to Spawn

Teleport a player back to The Rusted Flagon (spawn). Resets their interaction state.

Warp All Players to Spawn

Teleport every player back to The Rusted Flagon. Useful after updates or world resets.

Re-Seed World

Re-generate all world rooms from the lore seed. Existing rooms are refreshed to their original state.

Seed Demo Characters

Create demo test characters (Ari Quickstep & Marshal Vale) for smoke testing.

Reset World

Nuke all rooms and re-seed from lore YAML. Players can optionally be kept (reset to spawn with full HP/MP) or deleted entirely.
