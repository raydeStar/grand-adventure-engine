# P02 — Choose a License

**Category:** Polish
**Effort:** Tiny
**Touches:** `LICENSE`, `README.md`

## What

The README says "License not yet specified." Pick one and add it before people start forking.

## Options

| License | Good If... |
|---|---|
| **MIT** | You want maximum openness. Anyone can use, modify, distribute — even commercially. Most common for hobby/OSS projects. |
| **GPL-3.0** | You want to keep it open. Forks must also be open source. |
| **Apache-2.0** | Like MIT but with explicit patent protection. Good if you want corporate-friendly. |
| **AGPL-3.0** | Like GPL but covers server-side use too. If someone hosts it, they must share their changes. |

## Acceptance Criteria

- [ ] `LICENSE` file created in repo root with chosen license text
- [ ] README license section updated (replace "not yet specified" with license name and link)
- [ ] If using a standard license, use the full text from https://choosealicense.com/

## Notes

- For a self-hosted Discord bot/game engine, MIT or GPL-3.0 are the most common choices.
- This is a human decision — pick what feels right for how you want people to use your work.
